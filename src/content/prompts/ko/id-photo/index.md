## 프롬프트

```
업로드한 사진 속 인물을 기반으로 증명사진을 만들어줘.

- 얼굴 특징, 피부톤, 헤어스타일을 최대한 유지해줘
- 깨끗한 흰색 배경
- 여권/이력서용 증명사진 스타일
- 자연스럽고 부드러운 정면 조명
- 카메라를 정면으로 바라보는 무표정 또는 살짝 미소
- [색상]의 포멀한 셔츠/블라우스 착용
- 어깨까지 보이는 중앙 구도
- 얼굴에 선명한 초점, 스튜디오 품질 고해상도
```

## 사용 방법

1. ChatGPT (GPT-4o), Midjourney 등 이미지 생성이 가능한 AI 도구를 엽니다.
2. 본인 사진을 업로드합니다.
3. 위 프롬프트를 붙여넣기합니다.
4. `[색상]`에 원하는 옷 색상을 입력합니다. (예: 흰색, 연한 파란색, 검정)

## 팁

- 정면에서 찍은 선명한 사진일수록 결과가 좋습니다.
- 배경을 `연한 파란색 배경`으로 바꾸면 여권 사진 스타일이 됩니다.
- 실제 증명사진으로 사용하려면 생성된 이미지를 3x4 비율로 크롭하세요.

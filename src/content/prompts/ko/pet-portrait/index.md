## 픽사 스타일 프롬프트

```
업로드한 반려동물 사진을 픽사/디즈니 3D 애니메이션 캐릭터로 변환해줘.

- 동물의 털 색상, 무늬, 눈 색깔을 정확히 유지해줘
- 픽사 특유의 크고 반짝이는 눈, 과장된 표정으로 귀엽게 표현해줘
- 따뜻한 스튜디오 조명, 부드러운 배경 보케
- 캐릭터가 살아있는 것처럼 감정이 느껴지게 해줘
- 고품질 3D 렌더링 스타일
```

## 왕실 초상화 프롬프트

```
업로드한 반려동물 사진을 르네상스 왕실 초상화 스타일로 만들어줘.

- 동물의 얼굴 특징을 유지하면서 왕관과 화려한 왕실 의복을 입혀줘
- 클래식한 유화 텍스처와 금색 액자 포함
- 어두운 배경에 드라마틱한 조명
- 위엄 있고 당당한 포즈
- 16~18세기 유럽 왕실 초상화 느낌
```

## 사용 방법

1. ChatGPT (GPT-4o) 등 이미지 생성이 가능한 AI 도구를 엽니다.
2. 반려동물 사진을 업로드합니다.
3. 원하는 스타일의 프롬프트를 붙여넣기합니다.

## 팁

- 얼굴이 잘 보이는 정면 사진이 가장 좋은 결과를 줍니다.
- 픽사 스타일에 `영화 포스터 레이아웃`을 추가하면 포스터처럼 만들 수 있습니다.
- 왕실 초상화에서 `나폴레옹 스타일`, `엘리자베스 여왕 스타일` 등 구체적인 시대를 지정할 수 있습니다.

## 프롬프트

```
업로드한 사진을 스튜디오 지브리 스타일의 디지털 페인팅으로 변환해줘.

- 인물의 얼굴 특징과 표정을 유지하면서 지브리 특유의 화풍으로 표현해줘
- 부드러운 셰이딩, 회화적 텍스처, 따뜻한 색감을 사용해줘
- 배경도 지브리 영화의 한 장면처럼 몽환적이고 서정적으로 재해석해줘
- 전체적으로 평화롭고 감성적인 분위기로 연출해줘
- 지브리 애니메이션의 손그림 느낌을 살려줘
```

## 사용 방법

1. ChatGPT (GPT-4o) 등 이미지 생성이 가능한 AI 도구를 엽니다.
2. 변환하고 싶은 사진을 업로드합니다.
3. 위 프롬프트를 붙여넣기합니다.

## 팁

- 풍경이 포함된 사진이 더 드라마틱한 결과를 줍니다.
- 시간대를 추가하면 분위기가 달라집니다: `황금빛 석양 아래`, `달빛 아래`, `안개 낀 아침`
- 감정 키워드를 넣으면 톤이 바뀝니다: `평화로운`, `모험적인`, `쓸쓸한`, `즐거운`
- 지브리 요소를 추가해보세요: `빛나는 숲의 정령`, `떠다니는 등불`, `벚꽃 비`

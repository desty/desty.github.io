## Prompt

```
Turn the person in the uploaded photo into an action figure inside a plastic blister package box.

- Preserve the person's facial features, hairstyle, and body type as a figure
- The figure should be inside a clear plastic blister package
- Box header with figure name: "[NAME]"
- Box footer with title/description: "[JOB/TAGLINE]"
- Include mini accessories next to the figure: [ITEM1], [ITEM2], [ITEM3]
- Box background color: [COLOR]
- Realistic product photography style with studio lighting
```

## How to Use

1. Open an AI tool with image generation (e.g., ChatGPT GPT-4o).
2. Upload your photo.
3. Fill in the brackets in the prompt:
   - `[NAME]`: Figure name (e.g., DESTY)
   - `[JOB/TAGLINE]`: Description (e.g., Full-Stack Developer)
   - `[ITEM1~3]`: Accessories (e.g., laptop, coffee mug, keyboard)
   - `[COLOR]`: Box theme color (e.g., black, red)

## Tips

- Full-body photos work best for capturing body proportions.
- Customize accessories to match your hobbies or profession for personality.
- Add `anime style` for a cartoon-style figure.
- Add `vintage retro package` for a 90s toy vibe.
